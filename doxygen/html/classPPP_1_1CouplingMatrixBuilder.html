Detailed Description

detect item pair coupling in the target expensive processor and build a coupling matrix

the coupling matrix is more general than Geometry's adjacency matrix. this processor can be used to detect/filter potential coupling and skip item pair definitely not in coupling. parallel accessor need this information to coordinate coupling operation in parallel
