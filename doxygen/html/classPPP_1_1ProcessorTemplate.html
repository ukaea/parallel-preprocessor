Detailed Description

template<typename ProcessorType = Processor, typename ResultItemType = bool>
class PPP::ProcessorTemplate< ProcessorType, ResultItemType >

template + lambda to build a new processor inplace without declare a new processor type. this class will not be wrapped for Python, only used in C++, see example in ParallelAccessorTest.cpp myResultData is the place to save result data

Parameters
ProcessorTypea concrete process class it inherits from, in order to share data
ResultItemTypeprocessed result of VectorType<ResultItemType>, default to bool, will be saved to myOutputData a vector<ResultItemType>

TODO: add more template parameter to make a constructor() without parameter, not working lambda can capture the processor instance pointer or reference [&p] () {}

This template must derive from a concrete type Processor, not from a template parameter type "ProcessorType"
