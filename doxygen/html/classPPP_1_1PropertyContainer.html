Detailed Description

dynamic heterogeneous container to simulate dynamic property, based on std::any

APIs are compatible with STL associative container with STL iterator, but those APIs protected because Property type is only accessible by derived class Public access can only access pointer or value to the Property.data of specific type.

Data saved into std::any are shared_ptr<T>, to avoid large data copying around
